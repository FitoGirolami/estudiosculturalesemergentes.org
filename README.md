# Centro de Estudios Culturales Emergentes – Ecuador

Este repositorio contiene el código fuente del sitio web oficial  
https://estudiosculturalesemergentes.org/

## Objetivo

Documentar y analizar fenómenos culturales emergentes, con foco inicial en:
- Duendismo cultural en Baños de Agua Santa
- Tradición inventada y turismo
- Archivo de defensa cultural independiente

## Estructura

- `index.html`: página principal del sitio
- `/img`: recursos gráficos
- Otros documentos: próximos módulos de archivo y publicaciones.
